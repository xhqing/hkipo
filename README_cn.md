# 港股打新策略

基于数据驱动的港股IPO申购科学策略，针对2025年港交所新规后的市场环境优化。

[English](README.md)

---

## 背景

2025年8月4日，港交所IPO新规正式生效。约87%的新股选择机制B（无回拨、公开发售比例固定10%），一手中签率中位数从约50%断崖式跌至1%-2%。港股打新市场已从"盲打红利期"进入**"精选+执行纪律+多账户协同"**时代。

本项目提供一套系统化、量化的策略框架，帮助投资者在新规环境下科学决策。

---

## 策略概览

策略分为六个阶段：

| 阶段 | 核心主题 | 解决的关键问题 |
|------|---------|--------------|
| 1. 入场筛选 | 量化评分与一票否决 | 打不打？ |
| 2. 回拨与中签率分析 | 机制A vs B、中签率动态 | 机制如何影响中签？ |
| 3. 资金与申购策略 | 多账户、甲尾、乙头 | 打多少？ |
| 4. 卖出纪律 | 暗盘与首日规则 | 何时卖？ |
| 5. 行业赛道优先级 | 行业表现排名 | 哪些赛道优先？ |
| 6. 风险管理 | 资金红线、心理纪律 | 如何保护本金？ |

## 核心亮点

### 量化评分体系（7维度）

每只新股按7个加权维度评分（基石投资者、认购热度、国配倍数、行业赛道、保荐人/稳价人、估值合理性、老股转让）。**总分≥7分参与，<5分放弃，5-7分轻仓试水。**

### 一票否决机制

出现任一否决条件即自动放弃：主动回拨、无盈利路径、假基石、定价超上限15%、保荐人历史破发率>50%。

### 新规后核心洞察：甲尾反超乙头

机制B下投资者扎堆乙组，导致**甲尾中签率反而高于乙头**——这与新规前的经验完全相反。

### 多账户策略

一手中签率仅1%-2%时，小资金最有效的方式是**多账户各申购一手**（使用家人身份开户），中签率可达单账户的5-8倍。

### 严格卖出纪律

- 暗盘涨>20%：立即全部卖出
- 暗盘涨0%-5%：全部卖出（微涨新股首日砸盘概率高）
- 任何时刻破发：无条件市价卖出
- 首日从高点回撤>5%：立即卖出

## 数据基础

策略基于2021-2025年真实市场数据构建，包括：

- 港交所IPO新规影响分析（机制A/B采用比例）
- 甲组/乙组各层级中签率统计
- 回拨比例与破发率相关性（2020-2024年418个案例）
- 暗盘与首日表现相关性（正相关比例76%）
- 基石投资者占比与首日表现关系（25%-40%为"黄金区间"）
- 认购倍数"双峰曲线"（100-200倍为危险区间）
- 行业表现排名及代表案例
